# windwos Install consul

解压到 `E:\consul_1.11.1\consul.exe` ，并配置环境变量 `E:\consul_1.11.1`

仅开放测试环境启动命令：`consul agent -dev`

consul 自带 UI 界面，打开网址：[http://127.0.0.1:8500](http://127.0.0.1:8500) ，可以看到当前注册的服务界面
